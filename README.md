
# *GÖREV YÖNETİCİSİ*

Bu proje kullanıcının günlük, haftalık, aylık olarak yapacağı işleri planladığı küçük bir uygulamadır.

- Asp.Net Core Mvc 
- .Net Core 3.1  
-   Entity Framework Core SqlServer
-   Entity Framework Core Tools 
-   MSSql veri tabanı kullanarak 

yapılmıştır.
